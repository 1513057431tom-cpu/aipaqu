# 安全与合规设计

## 1. 信任边界

以下内容一律视为不可信：

- 用户输入和上传文件。
- 用户指定 URL 和第三方 API 响应。
- 抓取的网页正文、附件和页面脚本。
- 模型输入中的检索内容。
- 模型生成的文本、JSON、图表配置和工具参数。

## 2. 认证与授权

- 密码使用 Argon2id 哈希。
- 会话存放在 `httpOnly + secure + sameSite` Cookie 中。
- Cookie 名称使用 `__Host-` 前缀，生产环境必须启用 HTTPS。
- 登录接口限流并记录失败审计。
- 每个受保护 API 同时校验身份、角色和工作空间。
- `ADMIN` 管理凭据、模型、SMTP 和用户；`EDITOR` 不可读取加密秘密。

### CSRF 与 CORS

- 所有会改变状态的接口必须校验 CSRF Token，登录接口除外。
- CSRF Token 通过非 `httpOnly` Cookie 或 `/auth/csrf` 获取，提交时放入 `X-CSRF-Token` 请求头。
- CSRF Token 与会话绑定，注销或会话轮换时失效。
- CORS 只允许配置中的前端来源，禁止 `*` 与 `credentials=true` 同时出现。
- 预检请求只允许必要方法和头：`GET`、`POST`、`PATCH`、`DELETE`、`Content-Type`、`Idempotency-Key`、`X-CSRF-Token`。

### 权限矩阵

| 能力 | ADMIN | EDITOR |
| --- | --- | --- |
| 用户与角色管理 | 是 | 否 |
| 模型、Embedding、SMTP 配置 | 是 | 否 |
| 数据源凭据创建、更新、停用 | 是 | 否 |
| 创建和运行 ResearchBrief | 是 | 是 |
| 上传资料和数据集 | 是 | 是 |
| 编辑、审核和导出报告 | 是 | 是 |
| 查看加密秘密明文 | 否 | 否 |
| 查看审计日志 | 是 | 否 |

## 3. 凭据与秘密

- 网站账号、Cookie、API Token 和 SMTP 密码使用应用主密钥加密。
- API 永不返回秘密明文。
- 日志、事件、错误详情和 Prompt 中不得包含秘密。
- `.env` 不提交版本库；生产密钥必须独立配置和定期轮换。

## 4. URL 与 SSRF 防护

- 仅允许 `http` 和 `https`，生产默认要求 `https`。
- 解析全部 DNS 记录，拒绝回环、私网、链路本地、保留和云元数据地址。
- 每次重定向重新校验目标。
- 数据源默认使用域名白名单。
- 设置连接、读取、总请求超时和响应体大小上限。
- 浏览器采集容器不得访问内部管理网络。
- Browser Profile、代理和远程 CDP 地址都属于敏感配置，只能由 `ADMIN` 创建和停用。
- 远程浏览器连接必须校验来源、凭据引用和域名白名单，不允许任意连接用户输入的调试端口。
- 触发验证码、403、429、付费墙或访问控制页面时，任务应暂停并提示人工处理，不得自动生成绕过策略。

## 5. 上传与内容安全

- 限制扩展名、MIME、魔数、文件大小、页数、行数和压缩展开大小。
- 文件名由服务端生成，不使用用户输入作为路径。
- Office、PDF 和压缩文件解析在隔离任务中完成。
- 富文本输出进行白名单清洗，不执行脚本、事件属性和未知 URL。
- 上传文件先进入隔离区，解析成功后才进入可引用文件存储。
- 文件解析进程使用最小权限运行，不共享应用主密钥。
- 生产环境应接入恶意文件扫描；未配置扫描器时必须在健康检查或启动日志中明确标记为降级能力。
- 原始文件下载必须通过鉴权接口生成短期下载令牌，不暴露真实存储路径。

## 6. LLM 与 Agent 安全

- 网页中的“忽略以前指令”等文本只能作为资料内容。
- 工具权限由服务端代码定义，Prompt 不是权限边界。
- 模型输出必须经过 Schema 验证后才能进入下一节点。
- 模型不得直接执行 SQL、Shell、任意 Python 或浏览器脚本。
- 工作流限制调用次数、Token、超时、并发和返工次数。
- 报告将模型推断与来源事实显式区分。

## 7. 审计

记录但不泄露敏感内容：

- 登录、登出和失败登录。
- 用户、权限和配置变化。
- 凭据创建、更新、失效和测试。
- 采集任务创建、暂停、重试和取消。
- 模板发布和报告审核。
- 导出、下载和邮件发送。

### 日志脱敏

日志、事件和错误详情不得包含：

- 密码、Session ID、CSRF Token、API Key、Cookie、SMTP 密码。
- 完整身份证件、手机号、邮箱列表等高敏个人信息。
- 上传文件全文、报告全文和完整 Prompt。

允许记录摘要和引用：

- `sha256` 摘要。
- 文件大小、MIME、页数和行数。
- 资源 ID、任务 ID、节点 ID 和错误代码。

## 8. 合规边界

- 仅采集用户有权访问的内容。
- 遵守目标网站条款、robots 规则及适用法律。
- 不提供绕过验证码、付费墙或访问控制的能力。
- 对需要登录的数据源，由用户自行提供合法账号并确认使用权限。
- 支持按数据源配置保留期限和删除策略。
- 采集调试工作台只允许生成受审核的 `SiteAdapter` 配置，不允许执行用户提交的任意脚本。
